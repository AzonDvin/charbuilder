# Savage Worlds: Build Your Own Alien/Race Rules

*Based on Savage Worlds Deluxe Edition, p. 20–22*

---

## Overview

Races and cultures are built using a **point-based system**. Each race starts with a **free +2 Racial Ability** (equivalent to a Human’s Free Edge). Additional positive abilities must be balanced by an equal value of negative abilities.

**Example:** A +2 ability can be balanced by one –2 ability, or two –1 abilities.

---

## Building Rules

1. **Set the cap.** The GM chooses the maximum racial ability points allowed (usually **2–4**).
2. **Balance positives with negatives.** After the free +2, every positive ability must be offset by negative abilities of equal total value.
3. **Name abilities.** Give each ability a setting-appropriate name (e.g. “Born to the Saddle” for Riding d6).

## Quick Reference: Point Values


| Value | Typical Use                                                        |
| ----- | ------------------------------------------------------------------ |
| +3    | Hardy, Seasoned Edge, d8 attribute                                 |
| +2    | Flight, d6 attribute, Edge, Construct, Pace 10, Aquatic, etc.      |
| +1    | Low light/infravision, natural weapons, immunities, skill d6, etc. |
| –1    | Minor Hindrance, Pace 5, Charisma penalty                          |
| –2    | Major Hindrance, Pace 3, Toughness/Parry penalty                   |
| –3    | Hard attribute cap or double cost                                  |

## Example: Custom Star Wars Race

**Zabrak (simplified custom)**  

- Free +2 base  
- +1 Low light vision (+1)  
- +1 Resistant to fear (+1; +2 to resist Fear checks)  
- –1 Minor Hindrance: Stubborn (–1)  
- **Total: +2** (within cap)

**Aquatic species**  

- Free +2 base  
- +2 Aquatic (+2)  
- –1 Pace 5 (–1)  
- –1 Must return to water periodically (–1)  
- **Total: +2**

## Notes

- **Human baseline:** One free Edge ≈ +2 racial ability.
- **Unlisted abilities:** Assign a cost based on closest examples.
- **Flavor:** Prefer short, evocative names over long mechanical lists.
